Program Pendidikan & Layanan Kami

Paket A (Setara SD)

Program pendidikan kesetaraan tingkat Sekolah Dasar bagi masyarakat yang ingin merampungkan pendidikan dasar resmi.

Paket B (Setara SMP)

Program kesetaraan tingkat Sekolah Menengah Pertama untuk membekali peserta didik dengan wawasan dan ijazah resmi setara SMP.

Paket C (Setara SMA)

Program kesetaraan tingkat Sekolah Menengah Atas/Kejuruan untuk melanjutkan kuliah atau memasuki dunia kerja profesional.
